## 카테고리 가이드

- `dispatch` — agent 잘못 부름 / 부르지 않음
- `scope` — Scope 밖 파일 건드림 / Scope 안인데 놓침
- `chain` — 단위 경계 자동 chain / 잘못된 phase 전이
- `spec` — spec 미반영 / spec과 어긋난 plan
- `verify` — 검증 누락 / 잘못된 metric
- `style` — 4원칙 위반 (over-engineering 등)
